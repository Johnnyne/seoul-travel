# 首爾親子七日行程摘要

> 主要體驗請以 **GitHub Pages / 靜態站首頁（`index.html`）** 為主：那裡是完整的手機版互動行程。
> 這一頁則保留成 **GitHub 直接瀏覽時最實用的 Markdown 精簡版**，方便快速掌握最後定案。

## 旅程重點

- **日期**：2026/5/9（六）– 5/15（五）
- **成員**：2 位大人 + 1 位 4.5 歲孩子
- **住宿**：前 4 晚住江南 **Dormy Inn Seoul Gangnam**；後 2 晚住孔德 **GLAD Mapo**
- **節奏**：江南暖身 → 室內日 → Seoul Land → Everland → 文化日 → 購物日 → 返程
- **主軸**：不追求塞滿景點，而是讓一家三口玩得順、走得動、吃得舒服

## 為什麼改成 Seoul Land

原本的 **Lotte World** 已正式改為 **Seoul Land + 首爾大公園**。這個版本更適合 2 大 1 小的親子節奏，原因如下：

1. **設施更溫和**：對 4.5 歲孩子更友善，不必把時間花在不能玩的刺激設施旁。
2. **排隊通常較短**：體力耗損比較低，大人也不容易被拖垮。
3. **空間感更舒服**：搭配首爾大公園與 Elephant Train，整天的移動更有彈性。
4. **保留 Everland 的記憶點**：讓 Everland 成為唯一的高強度主題樂園主菜日，避免前半段連兩天硬仗。

## 最終日程

1. **Day 1 · 5/9（六）｜抵達 + 奉恩寺 / COEX 外圍散步**
   金浦入境後前往江南入住；如果精神還行，再到奉恩寺或 COEX 外圍走一圈，晚餐就近解決。
2. **Day 2 · 5/10（日）｜COEX 水族館 + Starfield Library + COEX Mall**
   以室內舒適日為主，水族館是核心，下午保留給拍照、逛街和休息。
3. **Day 3 · 5/11（一）｜Seoul Land + 首爾大公園**
   以 Kids Zone、Fantasy Land 和 Elephant Train 為重點，走親子樂園節奏，不追刺激設施。
4. **Day 4 · 5/12（二）｜Everland 主題樂園日**
   先動物區，再親子設施區；孩子累了就提早收工，不硬拼夜秀。
5. **Day 5 · 5/13（三）｜換飯店 + 景福宮 + 韓服 + 仁寺洞**
   上午從江南轉到孔德寄放行李，下午輕裝跑文化路線，晚上回麻浦 / 孔德吃飯。
6. **Day 6 · 5/14（四）｜聖水洞 + 首爾林：成熟系購物約會日**
   改成聖水洞 + 首爾林，因為比明洞少觀光壓力、比弘大少學生感，設計 / 生活風格店更集中，更適合兩位大人的成熟系購物與約會節奏。
7. **Day 7 · 5/15（五）｜睡晚一點 + 孔德早餐 + 金浦返程**
   最後一天只留早餐、收行李、AREX 去金浦；節奏更乾淨，也比較不會把返程變成趕場。

## 餐廳速查

- **江南 / COEX**
  - **Jiho Hanbang Samgyetang**：蔘雞湯溫和、適合抵達日補體力；招牌是 **蔘雞湯、糯米雞湯**。
  - **Yeongdonggeujip（영동그집）**：想吃一餐像樣韓牛可選這家；招牌是 **韓牛烤肉、牛排部位**。
- **Seoul Land / 果川**
  - **Dongsung Hall（동성회관）**：家常韓食穩定、不會太油膩；招牌是 **水煮豬肉片、韓式湯品**。
  - **Ppalgan Tteokbokki（빨간떡볶이）**：想快速補給時很實用；招牌是 **辣炒年糕、魚板、炸物**。
- **Everland / 龍仁**
  - **Dubu Madang（두부마당）**：手工豆腐料理清爽，親子友善；招牌是 **嫩豆腐鍋、豆腐火鍋**。
  - **Jajak Namu Galbi Yongin**：如果想把樂園日吃得更有儀式感；招牌是 **韓式烤排骨**。
- **景福宮 / 仁寺洞 / 孔德**
  - **Tosokchon Samgyetang（토속촌 삼계탕）**：景福宮周邊的經典蔘雞湯，穿韓服走完後很適合補一餐。
  - **Yukjeon Hoegwan（역전회관）**：孔德晚餐首選之一；招牌 **Bassak Bulgogi** 很有記憶點。
  - **Guldari Sikdang（굴다리식당）**：若想把最後一晚留在孔德附近舒服收尾很適合；招牌是 **泡菜鍋、辣炒豬肉、蛋捲**。
- **聖水 / 首爾林**
  - **Chamanda Seoul Forest（차만다 서울숲 본점）**：空間安靜、約會感強；招牌是 **Risotto、義大利麵、排餐**。
  - **Somunnan Seongsu Gamjatang（소문난성수감자탕）**：聖水在地老牌，逛街中段補體力很實用；招牌是 **감자탕、볶음밥**。
  - **Daesung Galbi（대성갈비）**：適合把 Day 6 晚餐吃得更像約會收尾；招牌是 **豬排骨、冷麵**。

## 快速提醒

- 如果 5/11 遇到明顯大雨，再考慮把 Seoul Land 改成室內備案；正常天氣下，這版比 Lotte World 更不重複，也更適合一家三口。
